# SouthNewSystem - نظام إدارة مؤسسة الجنوب الوثيق

نظام ERP متكامل مبني ليدعم اللغة العربية بالكامل (RTL) ويشمل إدارة المبيعات، المخزون، الموارد البشرية، المقاولات، والمهام.

## المميزات الجديدة (تمت إضافتها مؤخراً)

- **الموارد البشرية (HR)**: إدارة بيانات الموظفين، الرواتب، والمسميات الوظيفية.
- **إدارة المهام والمشاريع**: إنشاء وتعيين مهام للموظفين وربطها بالمشاريع والعقود.
- **الأرشيف والوثائق**: واجهة لإدارة المستندات والعقود (قيد التطوير لرفع الملفات).
- **تصميم عصري**: واجهة مستخدم محسنة وسهلة الاستخدام.

## التقنيات المستخدمة

- **Frontend**: React + Vite + Vanilla CSS Premium Design
- **Backend**: Node.js + Express + Prisma ORM
- **Database**: PostgreSQL (Supabase/Local)
- **Desktop**: Electron Wrapper

## طريقة التشغيل (سهلة وبسيطة)

### 1. التثبيت لأول مرة

افتح موجه الأوامر (Terminal) في المجلد الرئيسي للمشروع ونفذ الأمر التالي لتثبيت كافة المكتبات اللازمة دفعة واحدة:

```bash
npm run install:all
```

### 2. بدء تشغيل النظام

لتشغيل الخادم والواجهة الأمامية معاً بأمر واحد:

```bash
npm start
```

- **الواجهة الأمامية (Client)** ستعمل على: [http://localhost:3005](http://localhost:3005)
- **الخادم (Server)** سيعمل على: [http://localhost:5000](http://localhost:5000)

### 3. تطبيق سطح المكتب

لتشغيل النظام كتطبيق سطح مكتب:

```bash
npm run electron
```

## ملاحظات

- تأكد من وجود ملف `.env` في مجلد `server` يحتوي على بيانات الاتصال بقاعدة البيانات.
- تم تجميع أوامر التشغيل في الملف الرئيسي `package.json` لتسهيل الإدارة.
